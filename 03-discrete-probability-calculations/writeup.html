1 Introduction

Baseball is America’s pastime and its championship, the World Series, is a best-of-7 series between the winners of the American League and their National League counterparts. First played in 1903 between the Boston Americans and Pittsburgh Pirates, the World Series has seen several format changes throughout its 117 editions. No team has won the Fall Classic more than the New York Yankees with 27 World Series championships. For our blog’s sake, the head-to-head probability of the Yankees against the Braves World Series is below:

What is the Shortest Series Length for the Probability of the Braves Winning the World Series Given that PB = 0.55 is Greater than or Equal to 80%?

series_length <- function(x) {
  pnbinom((x-1)/2, (x+1)/2, 0.55)
}

for (j in seq(1,1000,2)) {
  if (series_length(j) >= 0.8){
    print(j)
    print(series_length(j))
    break
  }
}
## [1] 71

To answer the question of finding the shortest series length, I needed to create a function that would test each possible series of N length possible. In the series_length function I created above, it would use the pnbinom() to find the first value that would give us a greater than 80% chance. We found that the shortest series length so that P(Braves win World Series | PB = 0.55) is greater than or equal to 0.8 would be a best of 71 game series. The Braves would have an 80.17% chance of winning the World Series if it is a best-of-71.

What is the Shortest Series Length for the Probability of the Braves Winning the World Series Given that PB = x is Greater than or Equal to 80%?

q = seq(0.001,1,0.001)
v <- rep(NA, 500)
n_series_length <- function(n,p){
  pnbinom((n-1)/2, (n+1)/2, p)
}
for (i in 1:1000) {
  for (j in seq(1,1000,2)) {
    if (n_series_length(j, 0+0.001*i) >= 0.8){
      v[i] = j
      break
    }
  }
}
plot(q,v, 
     main = "Shortest Series so that (Win World Series Given P) >= 0.8",
     xlab = "Probability of the Braves Winning a Head-To-Head Matchup",
     ylab = "Series Length",
     type = "l")

To find the shortest series so that the Braves win the World Series given that PB = x I created a new function called n_series_length() that takes 2 parameters and runs them through the pnbinom() function. As we can see from the graph above, as the series length decreases from 1000, the odds of the Braves winning a head-to-head matchup increases.
